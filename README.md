# hokkaidoutaiken
北海道の「1日密着型リアル体験サービス」事業

あなたはプロのWebデザイナー兼トップマーケターです。
以下の条件をすべて満たした「実際に予約が入るランディングページ（LP）」をHTMLで作成してください。

【目的】
北海道の「1日密着型リアル体験サービス」の予約獲得（最重要）

【ビジネスモデル】
・1日1組限定（1〜4名）
・料金：50,000円 / 1組
・完全プライベート体験
・現地集合・現地解散
・1日同行型のローカル体験

【コンセプト】
観光ではなく「北海道のリアルな暮らしに入り込む1日」
→ 非日常ではなく“生活体験”
→ 「人に会いに行く旅」

【ターゲット】
・東京 / 神奈川 / 埼玉の20〜40代
・観光に飽きている層
・リアル・ローカル・体験重視
・SNS（TikTok・Instagram）ユーザー

【超重要：表現ルール】
以下は必ず守ること

・「送迎」という言葉は絶対に使わない
・「ツアー」という言葉も極力使わない
・「同行」「一緒に過ごす」「案内」などの表現を使う

▼ 正しい表現例
・「当日は移動を含めて1日ご一緒します」
・「状況に応じて柔軟にご案内します」
・「地元の暮らしを一緒に体験します」

【提供体験（3要素）】
※それぞれ必ず“具体的なシーン描写”を入れる

① 雪国生活体験（冬）
・朝6時 / 気温-10℃ / 雪かき
・ストーブで暖を取る生活
・地元の朝食

② 牧場体験
・早朝の牛の世話
・搾乳
・匂い・音・空気感の描写

③ ローカル飯
・観光客ゼロの店
・地元民との会話
・リアルな空気感

【重要：必ず入れる要素】
・時間 / 気温 / 五感（匂い・音・寒さ）
・体験後の感情変化（価値観が変わる等）
・ストーリー性

【構成（必須）】

① ファーストビュー
・キャッチコピー：
「観光じゃない。北海道の“暮らし”を体験する。」
・サブコピー（感情に刺さる）
・CTA（目立つボタン）

② 共感
・「観光地はどこも同じに感じる」など

③ 解決
・「ここでは生活に入ります」

④ 体験内容（カード形式）
・具体描写あり

⑤ 1日の流れ（重要）
・時系列で見せる（例：6:00 → 12:00 → 夜）
・リアルな1日をイメージさせる

⑥ ストーリー
・北海道在住の「大川」
・なぜやっているのか

⑦ 料金
・50,000円 / 1組（1〜4名）
・「人数が増えるほどお得感」

⑧ 限定性
・「1日1組限定」
・「枠がすぐ埋まる」など

⑨ 口コミ（3件）
・リアルで具体的に

⑩ FAQ（超重要）
以下を必ず含める：

Q. 移動はどうなりますか？
A. 当日は移動を含めて1日ご一緒します。状況に応じて柔軟に対応します。

Q. 1人でも参加できますか？
Q. どんな服装が必要ですか？
Q. 雨や雪の場合は？

⑪ CTA（複数配置）
・「今すぐ予約」

・LINE予約導線
